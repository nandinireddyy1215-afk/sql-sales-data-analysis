# SQL Sales Data Analysis Project

This project analyzes sales performance using SQL queries.

## Dataset
Superstore sales dataset containing order date, region, category, sales, and profit.

## SQL Analysis Performed

• Total Sales Calculation  
• Total Profit Calculation  
• Sales by Region  
• Sales by Category  
• Monthly Sales Trends  

## Example Query

SELECT Region, SUM(Sales) AS Region_Sales
FROM superstore_sales
GROUP BY Region;
